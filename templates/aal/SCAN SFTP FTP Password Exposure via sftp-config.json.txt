Terdeteksi aktivitas berbahaya berupa percobaan akses langsung ke file sensitif /sftp-config.json pada aplikasi web internal. File tersebut dikenal sering berisi konfigurasi SFTP/FTP termasuk kredensial dalam bentuk plaintext dan tidak diperuntukkan untuk akses publik. Aktivitas ini mengindikasikan upaya information disclosure reconnaissance yang bertujuan memperoleh kredensial atau informasi sensitif untuk mendukung serangan lanjutan. Meskipun tidak teridentifikasi kebocoran data pada sesi ini, aktivitas diklasifikasikan sebagai malicious attempt.

===============================

- Memastikan file sftp-config.json tidak ada atau tidak dapat diakses via web server.
- Melakukan audit direktori web root untuk memastikan tidak ada file konfigurasi sensitif yang terekspos.
- Menerapkan deny rule pada WAF / web server untuk file sftp-config.json, *.json
